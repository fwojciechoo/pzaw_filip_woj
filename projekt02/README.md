Projekt 02 - ksiega gosci

Opis************************

Prosta aplikacja Express.js - ksiaga gosci
Użytkownik mogą przeglądać wpisy innych osób i dodawać własne wiadomości.  


Uruchamiannie**************************

1. Sklonuj repozytorium i przejdź do folderu projektu.
2. Zainstaluj zależności:
```bash
npm install
```
3. Uruchom aplikację:
```bash
npm start
```
Otwórz w przeglądarce: http://localhost:8000

trasy**************************

 GET  /  Strona główna – lista wpisów 
 GET  /form  Formularz dodania wpisu 
 POST  /add  Dodanie wpisu i przekierowanie na stronę główną
